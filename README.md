Acoustic Project

The repository solves the inverse problem of the acoustic equation. 
COMPLETE



Files:

acoustic_model.py - Introduces the AcousticModel, ChebychevPropagator, and Detector classes. These classes allow solving the acoustic equation
                    with periodic boundary conditions, employing the Chebyshev expansion.
derivative_with_fft - FFT-based differentiation functions.
simple_wave_eq_solver.py - Solution of the wave equation utilizing the finite difference method.
simple_wave_eq_solver_fft - Solution of the wave equation utilizing FFT-based differentiation.
